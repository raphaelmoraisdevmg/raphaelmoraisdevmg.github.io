Assitente de automação de sistemas - NSM(hexion.inc) 11/2023 - atualmente

Atualmente atuo como Assistente de Automação na planta da Hexion Inc. (Curitiba/PR), alocado pela Terceira NPE, com foco na administração e manutenção da rede de controle do sistema DeltaV. Sou responsável por um ambiente composto por 18 workstations e 2 servidores, garantindo a estabilidade, segurança e disponibilidade do sistema de automação industrial. Minha atuação envolve a manutenção contínua do DeltaV, aplicação de atualizações recomendadas pelo Guardian, administração de backups e suporte direto à operação. Também trabalho no desenvolvimento de scripts em PowerShell e ajustes de Registro do Windows, além da modernização de lógicas antigas e desenvolvimento de soluções em VBA para o DeltaV e para planilhas em Excel, visando otimização de processos e maior confiabilidade operacional. Possuo experiência com preparação e integração de novas máquinas à rede, cotação técnica de equipamentos, manutenção preventiva e corretiva de workstations, comissionamento de instrumentos e gestão de ativos do ambiente DeltaV. Atuo ainda no acompanhamento de dosagens e eventos de processo, contribuindo para a continuidade e eficiência da planta. O ambiente é baseado integralmente em tecnologias Microsoft, utilizando atualmente Windows 10 LTSC IoT 2021 (21H2) e Windows Server 2022 Standard LTSC IoT. Tenho perfil técnico, organizado e orientado à solução de problemas, com forte foco em confiabilidade de sistemas industriais e melhoria contínua.

jogo Pedra, Papel e Tesoura-HTML & JS

Este projeto faz parte do curso Foundations do The Odin Project e tem como objetivo desenvolver o clássico jogo Pedra, Papel e Tesoura utilizando JavaScript. Inicialmente, o jogo é executado totalmente no console do navegador, sem interface gráfica, permitindo que o aluno foque na lógica de programação e na resolução de problemas. [theodinproject.com]

Ao longo do projeto, o desenvolvedor pratica a criação de funções, o uso de condicionais, a captura de entradas do usuário e a geração de escolhas aleatórias para o computador. Além disso, o projeto incentiva boas práticas como planejar com pseudocódigo, testar o código frequentemente e versionar o trabalho no GitHub, construindo uma base sólida para projetos futuros em desenvolvimento web.

Titulo2- HTML & CSS

O projeto Landing Page consiste na criação de uma página web completa a partir de um layout de design fornecido, utilizando HTML e CSS. O objetivo é reproduzir a estrutura visual geral do layout — como seções, alinhamentos e hierarquia de conteúdo — sem a necessidade de precisão absoluta em pixels, focando na construção do layout do zero. [theodinproject.com] Durante o desenvolvimento, o projeto reforça conceitos fundamentais de estrutura semântica em HTML, estilização com CSS e organização de layouts, além de incentivar o uso de boas práticas como pesquisa ativa, adaptação de conteúdo próprio e versionamento do código com Git e GitHub. Esse projeto marca um passo importante na transição do HTML básico para a criação de páginas visualmente estruturadas.

Receitas-HTML

O projeto Recipes tem como objetivo praticar os fundamentos de HTML por meio da criação de um site simples de receitas. Nele, o desenvolvedor constrói uma página principal que contém links para páginas individuais de receitas, utilizando apenas HTML, sem foco em estilo visual ou JavaScript. [theodinproject.com]

Durante o desenvolvimento, são aplicados conceitos essenciais como estrutura básica de páginas HTML, uso de títulos, parágrafos, listas, links e imagens. O projeto também introduz boas práticas iniciais de organização de arquivos e versionamento com Git e GitHub, servindo como o primeiro passo na construção de um portfólio em desenvolvimento web.
